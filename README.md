# LogicArt Dashboard - Power BI Project

## Overview
This project presents an interactive Power BI dashboard built to analyze business performance using the LogicArt dataset.

## Key Insights
- Profit and sales performance analysis
- Customer-level insights using Customer ID
- Impact of discount on profitability
- Quantity trends across different categories

## Files Included
- Power BI Dashboard (.pbix)
- Dataset (Excel)
- Dashboard Image (PNG)

## Tools Used
- Power BI
- Microsoft Excel

## Note
This is a beginner-level project focused on building foundational skills in data visualization and dashboard creation.
