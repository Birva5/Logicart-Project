# Serveto Dashboard - Power BI Project

## Overview
This project presents a Power BI dashboard created for Serveto, a 3PL logistics partner, to analyze sales, orders, 
and shipping performance for a premium customer.

## Objective
The goal of this dashboard is to provide insights into:
- Sales and profit performance
- Shipping trends and modes
- Customer and segment analysis

## Key KPIs (2021)
- Total Orders
- Total Sales
- Total Profit
- Total Quantity
- Average Discount
- Profitability
- Quantity for Standard Class (Accessories category)

## Dashboard Features
- Filters: Year, Region, State, Segment, Category, Sub-category
- Region-wise shipping distribution
- Category sales by shipping mode
- Trend of quantity by ship mode
- Segment-wise profit distribution
- Regional distribution of orders (Donut chart)

## Data
- Order data (2 years)
- Shipping data
- Data transformation performed before visualization

## Tools Used
- Power BI
- Excel

## Note
This project was created as part of a guided assignment to build foundational skills in data analysis and dashboard design.
